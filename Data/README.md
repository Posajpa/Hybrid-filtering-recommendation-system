### Input Files

#### 1. `filmtv_movies.csv`
Main database.

#### 2. `query_set.csv`
Query set, separated by ";".

#### 3. `user_id.csv`
User IDs.

#### 4. `utility_matrix.csv`
Utility matrix with sparsity.

### Output Files

#### 1. `out_cb.csv`
Main result, contains utility matrix with predicted values.

#### 2. `out_tk.csv`
Top 20 queries recommended for each user.
